# MINJAME

On-chain micro-lending protocol for Indonesian crypto users — built on Solana.

## Concept
First loan up to $10 USDC with no collateral. Access gated by wallet behavior. Every repayment builds a public on-chain credit score.

## Stack
- Smart Contract: Anchor (Rust)
- Frontend: Next.js
- Network: Solana Devnet

## Structure
- `/program` — Anchor smart contract
- `/app` — Next.js frontend
- `/tests` — Integration tests
